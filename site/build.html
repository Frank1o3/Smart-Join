Build & Publish

Build extension

  1. Install deps: npm install
  2. Transpile TypeScript and copy extension assets: npm run build:ext
  3. Resulting extension is in the dist/ folder for local loading into your browser.

Build the site (version injection)

  1. Run npm run build:site — this will read src/manifest.json and create site/version.html with the current version and release details.
  2. Preview locally using a static server, e.g. npx serve site or python3 -m http.server 8000 --directory site

Publish

This repository includes a GitHub Action that publishes the `site/` folder to the gh-pages branch whenever you push to `main` (or change the workflow). To use GitHub Pages, enable Pages on the repository and use the gh-pages branch or the recommended GitHub Pages settings.

Notes

  • If you fork this repo, update the homepage/support/privacy URLs in src/manifest.json to point to https://YOUR_USERNAME.github.io/Smart-Join/.
  • To change the GitHub Actions deploy behavior, edit .github/workflows/deploy-site.yml.
