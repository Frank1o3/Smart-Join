# Contributing to Smart Join

## Maintainer

This project is maintained by **[@Frank1o3](https://github.com/Frank1o3)**. I'm currently the only person working on it, so please be patient with response times!

## Contributing

### Reporting Issues

[Open an issue](https://github.com/Frank1o3/Smart-Join/issues/new) for bugs or feature requests. Use the [support form](https://frank1o3.github.io/Smart-Join/support.html) if you prefer.

### Pull Requests

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Test thoroughly
4. Submit a PR with a clear description

**Please note:** As a solo developer, PR reviews may take 1-2 weeks.

### Development Setup

```bash
npm install
npm run build:ext  # Build extension
npm run build:site # Build website
```

See [build instructions](https://frank1o3.github.io/Smart-Join/build.html) for details.

## Code Style

- Follow existing patterns in the codebase
- Use clear variable names
- Comment complex logic
- Test in multiple browsers when possible

## Questions?

Tag @Frank1o3 in issues or discussions. Response time is typically within a week.
